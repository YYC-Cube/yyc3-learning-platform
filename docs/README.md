## 技术栈

- **前端**: Next.js 15, React 19, TypeScript
- **样式**: Tailwind CSS, shadcn/ui
- **数据库**: MySQL 8.0
- **认证**: JWT, bcrypt
- **状态管理**: React Hooks, SWR
- **表单**: React Hook Form, Zod
- **图表**: Recharts
