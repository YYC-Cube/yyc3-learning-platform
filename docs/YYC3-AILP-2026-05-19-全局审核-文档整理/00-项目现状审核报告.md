### 2.3 存在风险的问题 🔴

| 风险                         | 位置                                                 | 影响  | 缓解措施                                                |
| ---------------------------- | ---------------------------------------------------- | ----- | ------------------------------------------------------- |
| **认证模块未使用 bcryptjs**  | lib/auth.ts import bcryptjs 但 package.json 无此依赖 | 🔴 高 | 确认 bcryptjs 在 dependencies 中（实际可能在 lockfile） |
| **images.unoptimized: true** | next.config.js                                       | 🔴 中 | 生产环境应启用图片优化，否则严重影响性能                |
| **JWT_SECRET optional**      | lib/env.ts JWT_SECRET optional                       | 🔴 高 | 生产环境 JWT_SECRET 必须必填，否则无认证安全            |
| **DB_PASS 默认空字符串**     | lib/env.ts DB_PASS default ''                        | 🔴 高 | 生产环境数据库密码不应有默认空值                        |
| **CSP仅限SVG**               | next.config.js contentSecurityPolicy                 | ⚠️ 中 | 图片CSP配置过于局限，非SVG图片可能被阻止                |

3. **添加 helmet 等安全头中间件** — Next.js middleware 中集成
4. **CORS 配置需审查** — API 路由应限制允许的来源
