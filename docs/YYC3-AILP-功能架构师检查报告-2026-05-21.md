## 🔍 二、业务逻辑分析

### 2.1 数据流架构

```
用户操作 → Page (Client) → API Route → Database/Static Data → Response
                ↓
         AuthContext / UserContext (localStorage 缓存)
                ↓
         useCourses / useUser (SWR-like hooks)
```

### 2.2 关键发现

#### ✅ 正确实现

| 功能          | 说明                                                                              |
| ------------- | --------------------------------------------------------------------------------- |
| **Auth 双层** | login/register 使用 PostgreSQL + bcrypt + JWT；前端 AuthContext 使用 localStorage |
| **安全体系**  | rate-limiter (内存) + security-headers + audit-log + input-validator              |
| **数据库**    | pg Pool 连接池 + 事务支持 + 优雅关闭                                              |
| **环境变量**  | Zod schema 验证 + 生产环境强制安全配置                                            |
| **错误边界**  | ErrorBoundary 组件 + error.tsx 全局捕获                                           |
| **加载状态**  | 8 个 loading.tsx 文件覆盖主要页面                                                 |

#### ⚠️ 架构性注意项

| 问题                    | 影响                                                          | 优先级 | 建议                  |
| ----------------------- | ------------------------------------------------------------- | ------ | --------------------- |
| **数据层双轨**          | courses/user API 使用内存模拟，login/register 使用 PostgreSQL | 高     | 统一为数据库          |
| **Auth 双轨**           | AuthContext 模拟登录 vs login API 真实登录                    | 高     | 统一为 JWT 认证流     |
| **全量客户端渲染**      | 20/24 页面 `"use client"`                                     | 中     | 拆分 Server/Client    |
| **零代码分割**          | 无 React.lazy / dynamic import                                | 中     | 按路由懒加载          |
| **Rate Limit 内存存储** | 单进程有效，多实例失效                                        | 中     | 迁移 Redis            |
| **缺失子页面 metadata** | SEO/社交分享缺失                                              | 中     | 添加 generateMetadata |
| **调试页面残留**        | debug-exam / test / test-edit-profile                         | 低     | 生产环境移除          |

#### 🔧 模拟数据统计

| 来源                 | 文件数 | 说明                            |
| -------------------- | ------ | ------------------------------- |
| `data/*.ts` 静态数据 | 11 个  | 课程/考试/题库/团队数据         |
| API 内存模拟         | 2 个   | courses/route.ts, user/route.ts |
| localStorage 模拟    | 2 个   | auth-context, AIWidgetContext   |
