### 🔍 单选题库

**文件位置**: [YYC3-AILP-理论-单选题/单选题.md](YYC3-AILP-理论-单选题/单选题.md)

#### 📋 题目示例

**第1题**

- **题干**: $\epsilon$差分隐私参数大小和隐私保护的关系是什么？
- **选项**:
  - A. 较小的$\epsilon$值意味着更强的隐私保护
  - B. 以上都不对
  - C. 较大的$\epsilon$值意味着更强的隐私保护
  - D. $\epsilon$值与隐私保护无关
- **答案**: A
- **解析**: 在差分隐私中，$\epsilon$是隐私预算，$\epsilon$越小，隐私保护程度越强；$\epsilon$越大，隐私保护程度越弱。

**第2题**

- **题干**: 在合规检查制度管理上，检查企业的人力制度应关注
- **选项**:
  - A. 数据安全管理部门或岗位是否拥有检查所有数据的权限
  - B. 数据安全管理部门或岗位的专业知识和技能
  - C. 数据安全管理部门或岗位离职率
  - D. 数据安全管理部门或岗位的培训记录
- **答案**: A
- **解析**: 数据安全管理部门的核心职责是拥有检查所有数据的权限，确保数据安全和合规性。

interface Question {

### 🎯 自动组卷算法

```typescript
class ExamGenerator {
  generateExam(
    category: string,
    difficulty: number,
    questionCount: number,
    timeLimit: number
  ): Exam {
    // 1. 按类别和难度筛选题目
    const questions = this.questionBank
      .filter((q) => q.category === category && q.difficulty === difficulty)
      .slice(0, questionCount);

    // 2. 随机打乱题目顺序
    const shuffledQuestions = this.shuffleArray(questions);

    // 3. 计算总分
    const totalScore = questions.reduce((sum, q) => sum + q.score, 0);

    return {
      id: this.generateId(),
      questions: shuffledQuestions,
      totalScore,
      passScore: Math.floor(totalScore * 0.6),
      timeLimit,
      category,
      difficulty,
    };
  }

  private shuffleArray<T>(array: T[]): T[] {
    const shuffled = [...array];
    for (let i = shuffled.length - 1; i > 0; i--) {
      const j = Math.floor(Math.random() * (i + 1));
      [shuffled[i], shuffled[j]] = [shuffled[j], shuffled[i]];
    }
    return shuffled;
  }
}
```
