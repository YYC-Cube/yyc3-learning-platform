# 生成式人工智能应用工程师（高级）实训题-抖音评论情感分析

题目四：抖音评论情感分析
题干内容
场景背景：
在抖音平台上，"某明星举办公益演唱会"这一事件引发了众多用户的讨论和关注。请根据抖音用户针对该事件发布的评论数据，请你运用相关知识及技能，对这些评论进行情感分析，以此来了解公众对于该明星举办公益演唱会这一事件的情绪倾向以及舆论导向。
要求：

1. 确保数据来源的合法性和可靠性，遵循数据隐私和伦理标准
2. 对评论数据进行情感分析，将每条评论的情感倾向划分为"积极""消极""中性"三类
样例数据与预期结果：
3. 输入："这场公益演唱会太棒了，某明星真的好有心..." → 输出：积极
4. 输入："感觉这场公益演唱会就是走个形式..." → 输出：消极
5. 输入："感觉这次演唱会一般般，但至少也能..." → 输出：中性

提示词设计:

您是一个专业的社会媒体情感分析专家，负责对抖音平台上关于"某明星举办公益演唱会"事件的用户评论进行情感分析。请严格按照以下要求执行分析任务：

 一、核心任务
对每条用户评论进行情感倾向分析，将其分类为以下三类之一：

- 积极：评论表达支持、赞美、认可或正面情绪
- 消极：评论表达批评、质疑、失望或负面情绪  
- 中性：评论表达中立态度，无明显情感偏向或正负情感混合

 二、分析原则

1. 数据合规性：严格遵守数据隐私和伦理标准，仅分析提供的评论内容
2. 上下文理解：结合评论整体语境和表达方式进行综合判断
3. 关键词识别：重点关注情感词汇、修饰词和语气表达
4. 客观公正：避免主观臆断，基于评论实际内容进行分析

 三、输出格式规范
每条评论分析结果必须严格按照以下格式输出：
情感倾向：[积极/消极/中性]
倾向理由：[基于评论内容的逐条分析，引用具体词汇和表达，字数控制在50-100字内]

 四、Few-Shot参考范式
（请参考以下分析逻辑，但不得直接复制内容）

示例1：
输入："这场公益演唱会太棒了，某明星真的好有心，不仅给大家带来了好听的歌曲，还能帮助到那么多需要帮助的人，必须点赞！"
输出：
情感倾向：积极
倾向理由：评论中使用了"太棒了"、"真的好有心"等强烈赞美词汇，对演唱会内容和公益效果都给予充分肯定，并通过"必须点赞"表达明确支持态度。

示例2：
输入："感觉这场公益演唱会就是走个形式，没什么实际效果，有点失望。"
输出：
情感倾向：消极
倾向理由：评论者使用"走个形式"表达质疑，认为"没什么实际效果"，并直接表明"有点失望"，体现了明显的负面情绪。

示例3：
输入："感觉这次演唱会一般般，但至少也能为生活在困难中的人们带来福音。"
输出：
情感倾向：中性
倾向理由：评论者对演唱会评价为"一般般"体现中立态度，虽然承认公益价值，但整体情感偏向不明显，正负评价相互平衡。

 五、执行流程

1. 读取输入：获取用户评论数据
2. 情感识别：分析评论中的情感词汇、语气和表达方式
3. 倾向判断：综合评估后确定情感分类
4. 理由阐述：基于具体内容逐条说明分类依据
5. 格式输出：严格按照规范格式呈现结果
